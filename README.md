# אלוף החלל 🚀

**אלוף החלל** הוא משחק דפדפן מהנה וממכר שנבנה ב-JavaScript, HTML ו-CSS. צאו למסע בין כוכבים, אספו אוצרות קוסמיים והוכיחו שאתם הטייסים המיומנים ביותר ביקום!

---

## 🎮 איך משחקים?

* **תנועה:** השתמשו בעכבר כדי להזיז את החללית שלכם במרחב.
* **המטרה:** אספו את היהלומים הצהובים והכחולים (💎) כדי לצבור נקודות ולשבור שיאים.
* **זהירות:** אל תיגעו במוקשים האדומים (☄️) – נגיעה אחת והמשחק נגמר!
* **שלבים:** ככל שתצברו יותר נקודות, תעלו לשלבים מתקדמים יותר שבהם המשחק הופך למהיר ומאתגר יותר.

---

## ✨ פיצ'רים בולטים

* **מערכת שלבים אינסופית:** רמת הקושי עולה ככל שמתקדמים.
* **ניהול שיאים:** המשחק שומר את השיא הכי גבוה שלכם לפי שם משתמש.
* **חנות שדרוגים:** (בקרוב/קיים) אפשרות לרכוש מגנים ושדרוגים בעזרת הנקודות שנאספו.
* **עיצוב:** חוויית משחק חלקה עם אפקטים של ניאון ופיצוצים.

---

## 🛠️ טכנולוגיות

המשחק נבנה ללא ספריות חיצוניות (**Vanilla JavaScript**), מה שהופך אותו למהיר מאוד, קליל וקל להרצה בכל דפדפן מודרני באמצעות HTML5 Canvas.
